# Лабораторная работа №1
## Знакомство с Java и базовые алгоритмы

---

## Описание

Данная работа является вводной и направлена на освоение базового синтаксиса языка Java, работу с командной строкой и виртуальной машиной Java (JVM). В процессе выполнения работы реализованы две программы, демонстрирующие применение циклов, условных операторов и методов для решения задач поиска простых чисел и проверки палиндромов.

---

## Ключевые концепции

- **Компиляция и выполнение**: Процесс преобразования исходного кода Java в байт-код и его выполнение на JVM
- **Классы и методы**: Организация кода в классы с использованием статических методов для выполнения операций
- **Циклы**: Использование цикла `for` для итерации по диапазонам чисел и символам строк
- **Условные операторы**: Применение условий `if-else` для ветвления логики программы
- **Операторы**: Использование оператора остатка (`%`) для проверки делимости и логических операций
- **Методы и параметры**: Передача данных методам и возврат результатов с использованием разных типов возврата (`boolean`, `String`)
- **Работа со строками**: Методы `charAt()`, `length()` и `equals()` для работы с текстовыми данными

## Структура проекта

Проект состоит из двух программ, каждая решающая отдельную задачу:

### **Primes.java — Поиск простых чисел**

Программа для определения и вывода всех простых чисел в диапазоне от 2 до 100.

- **Метод main(String[] args)**: Точка входа программы, содержит цикл для перебора чисел от 2 до 100 и вывод простых чисел
- **Метод isPrime(int num)**: Определяет, является ли переданное число простым
  - Использует цикл для проверки делимости на числа от 2 до √num
  - Возвращает `false` при обнаружении делителя
  - Возвращает `true`, если делителей не найдено

**Оптимизация**: Условие `i*i <= num` значительно ускоряет алгоритм, так как если число делится на делитель больший чем √num, то оно должно делиться и на число меньшее чем √num.

### **Palindromes.java — Проверка палиндромов**

Программа для определения, является ли введенная строка палиндромом (словом, читаемым одинаково в обе стороны).

- **Метод main(String[] args)**: Обрабатывает аргументы командной строки и проверяет каждую строку на соответствие критериям палиндрома
- **Метод reverseString(String word)**: Переворачивает строку в обратном порядке
  - Итерирует символы от конца строки к началу с помощью цикла `for`
  - Использует метод `charAt(int index)` для получения символа по индексу
  - Конкатенирует символы в новую строку
- **Метод isPalindrome(String word)**: Проверяет, является ли слово палиндромом
  - Вызывает `reverseString()` для получения перевернутой версии
  - Использует метод `equals()` для сравнения оригинальной и перевернутой строк
  - Возвращает `true`, если строки идентичны

## Содержание заданий

### Задание 1: Поиск простых чисел (Primes.java)

**Описание**: Программа находит и выводит все простые числа в диапазоне от 2 до 100 включительно.

**Реализованный функционал**:
- Создан класс `Primes` с методом `main()`, который перебирает числа от 2 до 100
- Реализован метод `isPrime(int num)` для проверки простоты числа
- Использован оператор модуля `%` для определения делимости
- Вывод найденных простых чисел в консоль

**Алгоритм проверки простоты**:

```java
public static boolean isPrime(int num) {
    if (num < 2) return false;  // Числа меньше 2 не являются простыми
    for (int i = 2; i*i <= num; i++) {  // Проверка до квадратного корня
        if (num % i == 0) {
            return false;  // Найден делитель
        }
    }
    return true;  // Делителей не найдено
}
```

**Результаты**: Программа выводит последовательность простых чисел: 2, 3, 5, 7, 11, 13, ... 97

**Команды для выполнения**:
```bash
javac Primes.java
java Primes
```

### Задание 2: Проверка палиндромов (Palindromes.java)

**Описание**: Программа определяет, являются ли введенные строки палиндромами (словами, читаемыми одинаково в обе стороны).

**Реализованный функционал**:
- Создан класс `Palindromes` с методом `main()`, обрабатывающим аргументы командной строки
- Реализован метод `reverseString(String word)` для разворота строки
- Реализован метод `isPalindrome(String word)` для проверки палиндрома
- Использованы методы работы со строками: `charAt()`, `length()`, `equals()`
- Вывод результата проверки каждой строки

**Алгоритм разворота строки**:

```java
public static String reverseString(String word) {
    String reversed = "";
    for (int i = word.length() - 1; i >= 0; i--) {  // От конца к началу
        reversed += word.charAt(i);  // Добавление символа в новую строку
    }
    return reversed;
}
```

**Алгоритм проверки палиндрома**:

```java
public static boolean isPalindrome(String word) {
    String reversed = reverseString(word);
    return word.equals(reversed);  // Сравнение с использованием equals()
}
```

**Примеры входных данных и ожидаемые результаты**:
- `madam` → palindrome (тем же читается в обратном порядке)
- `racecar` → palindrome
- `apple` → not palindrome
- `kayak` → palindrome
- `song` → not palindrome
- `noon` → palindrome

**Команды для выполнения**:
```bash
javac Palindromes.java
java Palindromes madam racecar apple kayak song noon
```

**Вывод программы**:
```
madam-> palindrome
racecar-> palindrome
apple-> not palindrome
kayak-> palindrome
song-> not palindrome
noon-> palindrome
```

## Важные замечания

- **Имя файла и класса**: Имя исходного файла Java должно совпадать с именем публичного класса. Например, класс `Primes` должен находиться в файле `Primes.java`

- **Метод main**: Сигнатура метода `main()` должна быть точной: `public static void main(String[] args)`, иначе программа не запустится

- **Компиляция и выполнение**: Сначала необходимо скомпилировать программу (создаст файл `.class`), а затем выполнить её через JVM

- **Аргументы командной строки**: Для передачи данных в программу используются аргументы командной строки после имени класса

- **Сравнение строк**: Для сравнения строк необходимо использовать метод `equals()`, а не оператор `==`
